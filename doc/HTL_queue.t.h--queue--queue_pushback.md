# `Queue_PushBack`
<small>*Function* - **modifiers**</small>  
**Synopsis**

```cpp
ContentType* Queue_PushBack(Queue* q)
```

Creates room for an element at the back


## Return Value


the allocated entry on success (that is `Back()` ),
`NULL` on failure.
It fails, when the queue is full
