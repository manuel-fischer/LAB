# `HTL_MEMBER`
**Synopsis**

```cpp
token HTL_MEMBER(name)
```

Generate an identifier in the current namespace.
The current namespace is specified by the current value
of the parameter `NAME`


## Parameter
**`name`** &#8213; the member name
