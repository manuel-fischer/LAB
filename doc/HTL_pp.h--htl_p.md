# `HTL_P`
<small>*Macro Definition*</small>  
**Synopsis**

```cpp
object HTL_P(token name)
```

Get a parameter of the current parameter set.
The parameter set is specified by the current token in `HTL_PARAM`


## Parameter
**`name`** &#8213; the name of the parameter
